# medical-image-reconstruction

The game will use strategy like elements from games like WarCraft 3, Command and Conquer, and Supreme Commander in order to deliver a gamified way of researching and learning about bacteria. The bacteria featured in the game will be 3D models created from the 2D images from a microscope. </br>

The gameplay is setup to feature a main character the player can click on and move around, the player can only build cells in the placement grid around the player. </br> </br>
## **Cell Building:**
  - These are the 3D models of the bacteria, that can be placed in the game map. Cells cost the main game resource to place.
  - Only a certain number of cells (equal to the slots in the top right corner) can be placed. In the shop tab the player can spend the currency to increase their slots.
  - These act as the player's "base", and provide passive income used as a method for scoring.
  - They can be upgraded through the external upgrade screen- where player's must correctly annotate and label parts of the bacteria, these labels are scored based on accuracy and after a certain score threshold is met the bacteria is upgraded to a higher rarity in the game and improves the building's stats and passive income.</br> </br>

# **Control Scheme (WIP)** 
[WASD]                     -> Pan camera </br>
[MOUSE SCROLL WHEEL]       -> Zoom in/ out </br>
[RIGHT CLICK + MOUSE MOVE] -> Rotate camera left/ right </br>
[LEFT CLICK]               -> Select _Character_ OR _Cell_ </br>
[LEFT CLICK]               -> (With _Moveable Troop_ selected) Move to destination
